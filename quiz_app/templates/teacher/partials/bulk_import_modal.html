CSV Format Requirements:

Your CSV file must contain the following columns:

  • full_name (required) - Teacher's full name
  • contact_number (required) - Contact number
  • gender (optional) - M or F
  • email (optional) - Teacher's email address
  • department (optional) - Department name
Accepted formats: CSV, Excel (.xlsx, .xls)
Applied if department column is empty in CSV
Note: Duplicate staff IDs will be automatically generated. Make sure all required fields are filled in the CSV file.
