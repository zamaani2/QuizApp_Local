File Format Requirements

Required Column: question_text

Optional Columns:

  • question_type - multiple_choice, true_false, short_answer, fill_blank, essay (default: multiple_choice)
  • marks - Points for correct answer (default: 1)
  • difficulty - easy, medium, hard (default: medium)
  • correct_answer - For multiple_choice: choice index (1, 2, 3...) or "1,2" for multiple correct. For true_false: "true" or "false"
  • choice1 to choice10 - Answer choices for multiple_choice questions
  • explanation - Explanation shown after answering

Tip: CSV is simpler and easier to edit. Excel provides better formatting but requires Microsoft Excel or similar software.
