CSV Format Requirements:

Your CSV file should contain student data. After uploading, you'll be able to map your CSV columns to the system fields. Download the template below to see the correct format with example data.

  • Required fields: Full Name, Gender
  • Optional fields: Admission Number (auto-generated if not provided), Date of Birth, Parent Contact, Admission Date, Email
  • Date format: YYYY-MM-DD (e.g., 2024-01-15)
  • Gender: M or F
  • Admission Number: If not provided, a unique admission number will be automatically generated
Accepted formats: CSV, Excel (.xlsx, .xls)
